# AR Group ERP Starter

ئەم پڕۆژەیە وەشانی یەکەمی سیستەمی ERP/POS ـە بۆ AR Group.

## چی تێدایە؟
- Login
- Dashboard
- Products / Stock
- Sales / Invoice
- Purchases
- Customers
- Suppliers
- Expenses
- Reports
- Smart Summary rule-based
- SQLite database

## چۆن بەکاری بهێنیت؟

1. Python دابەزێنە.
2. ئەم فرمانانە لە Terminal بنووسە:

```bash
pip install flask
python app.py
```

3. لە browser بچۆ:

```text
http://127.0.0.1:5000
```
